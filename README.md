# AnkiTrans - 划词制卡助手

**AnkiTrans** 是一款专为 Anki 用户打造的浏览器扩展，旨在帮助您在浏览网页时，以最快速度将生词捕捉并同步到 Anki 牌组中。它无缝集成了必应词典的权威数据，为您自动生成包含音标、释义、例句等丰富信息的精美卡片。

## ✨ 核心功能

* **⚡️ 极速制卡**：选中任意网页文本，通过右键菜单或快捷键 (`Ctrl+Shift+A`) 即可提取单词信息并制作卡片。
* **📖 丰富数据**：自动抓取 **必应词典** 的权威数据，包含：
  * 🇺🇸 / 🇬🇧 美式与英式双语发音（支持直接试听）
  * 📚 详细的中英文释义与词性标注
  * 📝 精选双语例句与常用搭配
  * 🔄 单词变形信息（复数、时态等）
* **🎨 精美模板**：内置精心设计的 `AnkiTrans` 专属卡片模板，完美适配桌面端与移动端，自动支持夜间模式。
* **🔄 无缝同步**：基于 `AnkiConnect` 与本地 Anki 直连，无需繁琐的导出导入。
* **🛠 便捷管理**：
  * **一键建牌**：直接在扩展弹窗中创建新的 Anki 牌组，无需切换应用。
  * **所见即所得**：提供真实的卡片预览功能，在保存前确认内容。
  * **智能缓存**：优化数据加载体验，减少重复请求。

## 📋 前置要求

在使用本扩展前，请确保您的环境满足以下条件：

1. **Chrome 浏览器** (或 Edge 等 Chromium 内核浏览器) - 版本 88 或更高。
2. **Anki 桌面版** - [下载 Anki](https://apps.ankiweb.net/)。
3. **AnkiConnect 插件** - 必须安装此插件以允许扩展与 Anki 通信。
    * 插件代码：`2055492159`
    * 安装方法：打开 Anki -> `工具` -> `插件` -> `获取插件` -> 输入代码并确认 -> **重启 Anki**。

## 🚀 安装说明

1. 下载最新的 release 压缩包 (例如 `AnkiTrans_v1.1.0.zip`) 并解压。
2. 打开 Chrome 浏览器的扩展管理页面：`chrome://extensions/`。
3. 开启右上角的 **“开发者模式”** 开关。
4. 点击左上角的 **“加载已解压的扩展程序”**。
5. 选择解压后的文件夹即可完成安装。

## 📖 使用指南

### 1. 初始化设置

* 点击浏览器右上角的 AnkiTrans 图标。
* 根据引导完成初始化（如果未检测到 Anki，请检查 AnkiConnect 是否运行）。
* 在主界面选择一个**目标牌组**（支持直接新建）。

### 2. 划词制卡

* **方法一（推荐）**：选中网页中的单词或短语，按下快捷键 **`Ctrl + Shift + A`**。
* **方法二**：选中文字后点击鼠标右键，选择 **“AnkiTrans: 添加到 Anki”**。

### 3. 使用技巧

* **卡片预览**：默认开启预览模式，制卡时会弹出悬浮窗显示卡片效果，点击确认后保存。您可以在扩展选项中关闭预览，实现“静默秒存”。
* **发音试听**：预览界面的音标旁有发音按钮，点击即可试听。

## ⚙️ 字段说明

AnkiTrans 会自动创建名为 `AnkiTrans` 的笔记类型，包含以下字段：

| 字段名 | 说明 |
| :--- | :--- |
| **Word** | 单词原文 |
| **Phonetic** | 美式/英式双语音标（含发音文件引用） |
| **Translation** | 主要中文释义与词性 |
| **Inflections** | 单词变形（复数、过去式等） |
| **Example** | 精选双语例句 |
| **Collocations** | 常见搭配 |
| **Synonyms** | 同义词/反义词 |
| **DomainDefs** | 网络/领域释义 |
