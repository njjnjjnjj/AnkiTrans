# AnkiTrans

一个 Chrome 浏览器扩展，可以将选中的单词或短语通过 AnkiConnect 推送到 Anki 牌组，实现沉浸式语言学习。

## ✨ 功能特点

- 🔤 **文本选择** - 选中网页中的任意文字快速添加
- 🌐 **自动翻译** - 支持多种语言翻译（默认翻译为简体中文）
- 📚 **Anki 集成** - 通过 AnkiConnect 直接创建 Anki 卡片
- ⚡ **快捷操作** - 右键菜单或键盘快捷键快速添加

## 📋 前置要求

1. **Chrome 浏览器** - 版本 88 或更高
2. **Anki 应用** - [下载 Anki](https://apps.ankiweb.net/)
3. **AnkiConnect 插件** - 在 Anki 中安装（插件代码：`2055492159`）

### 安装 AnkiConnect

1. 打开 Anki
2. 点击 `工具` → `插件` → `获取插件`
3. 输入代码 `2055492159` 并点击确定
4. 重启 Anki

## 🚀 安装扩展

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `AnkiTrans` 文件夹

## 📖 使用方法

### 方法一：右键菜单

1. 在网页中选中单词或短语
2. 右键点击，选择「添加到 Anki」
3. 等待翻译完成，卡片自动添加到 Anki

### 方法二：快捷键

1. 选中文字后按 `Ctrl + Shift + A`
2. 卡片自动创建

### 方法三：弹出窗口

1. 点击扩展图标打开弹窗
2. 在输入框中输入内容
3. 点击「添加」按钮

## ⚙️ 设置选项

在弹出窗口中可以配置：

- **目标牌组** - 选择卡片添加到哪个 Anki 牌组
- **翻译语言** - 选择翻译的目标语言

## 🔧 故障排除

### 连接失败

- 确保 Anki 已打开并运行
- 确保 AnkiConnect 插件已安装
- 检查 AnkiConnect 设置中的端口是否为 8765

### 翻译失败

- 检查网络连接
- 部分网站可能限制翻译 API 请求
